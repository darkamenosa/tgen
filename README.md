# tgen

A simple tool generates simple front-end app structure, built with webpack 4, boostrap, jquery.

## Getting Started

Install: 
```
npm install -g tgen
```

Run it:
```
mkdir example
cd example

tgen .
``` 

Output directory structure:
```
├── about
│   ├── about.css
│   └── about.js
├── common-import.js
├── home
│   ├── home.css
│   └── home.js
├── package.json
└── webpack.config.js
```

### Prerequisites

- NodeJS 6+
